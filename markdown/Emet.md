##

#### Un comando para poder agregar elementos dentro de una lista pero con diferentes atributos en cada uno.

```HTML
<!-- nav>ul>li*5{elemento$} -->
    <nav>
      <ul>
        <li>elemento1</li>
        <li>elemento2</li>
        <li>elemento3</li>
        <li>elemento4</li>
        <li>elemento5</li>
      </ul>
    </nav>
```
